## What this text is

"Chaman ki Shri Durga Stuti" (चमन की श्री दुर्गा स्तुति), a devotional
compilation (a Hindi-verse rendering of themes from the Durga Saptashati,
plus Durga Kavach, Mangala Stotra, Argala Stotra, Kilak Stotra, Maha Chandi
Stotra, Maha Kali Stotra, Maha Lakshmi Stotra, Santoshi Maa Stotra, Bhagwati
Naam Mala, Nav Durga Stotra, Annapurna Bhagwati Stotra, and an aarti) by
**Chaman Lal Bhardwaj "Chaman"** of Amritsar, published by Brij Mohan
Bhardwaj Pustakalaya / Mahamaya Chaman Mandir, Chheharta, Amritsar. This is a
**modern (20th-century) devotional work**, not an ancient/classical Sanskrit
text, closer in kind to the platform's other "hold" items (Santoshi Mata,
Khatu Shyam, Nangli Sahib) than to public-domain stotras like Ram Raksha or
Aditya Hridayam.

## Why this was rebuilt

The original capture (`RAW_devanagari_partial.txt`, kept in this folder for
the record) came from archive.org's automated OCR (`_djvu.txt`) of a scanned
print copy. Internet Archive's own OCR confidence stats for this item are
weak (a large share of words in low-confidence buckets), and this showed up
as real errors: for example दुर्या where the text should read दुर्गा (Durga),
and गान्ति where it should read शान्ति (Shanti). That was caught by direct
inspection, not assumed.

Given that, the OCR file was abandoned as a source entirely rather than
partially corrected. The full text was re-fetched from **anandeshwari.in**,
which hosts the same compilation as typed (born-digital) HTML across ~24
individual chapter/section pages, not a picture scan. Every section in
`RAW_devanagari.txt` was fetched directly from its own page on that site and
is verbatim, nothing reconstructed from memory or summary. WordPress site
chrome (navigation menus, "Recent Posts," category/tag footers, comment
threads, share buttons, cookie-consent boilerplate) was stripped out during
compilation; the devotional text itself was not altered.

  स्तोत्र नमस्कार (full), कीलक स्तोत्र (full), विनम्र प्रार्थना
- All 13 chapters (अध्याय 1 through 13), full
- महा चण्डी स्तोत्र, महा काली स्तोत्र, नमन प्रार्थना, मां जगदम्बे जी की
  आरती, महा लक्ष्मी स्तोत्र, श्री संतोषी मां स्तोत्र, श्री भगवती नाम माला,
  श्री चमन दुर्गा स्तुति के सुन्दर भाव, अन्नपूर्णा भगवती स्तोत्र, all full

Nothing from the site's table of contents is missing. The old
`RAW_devanagari_partial.txt` (OCR-derived, partial, contains known errors) is
kept in this folder only as a record of the original capture; it should not
be used as source text going forward. `RAW_devanagari.txt` supersedes it.

## Transliteration

**None found, still.** This is not on sanskritdocuments.org at all, it's a
regional modern devotional book digitized on Internet Archive and reproduced
on anandeshwari.in, and no Roman-transliteration edition turned up in
search. A transliteration would need to be produced from scratch, same as
the project's own practice for other items (per RUNBOOK_lyrics.md), not
